# 프로젝트명: 댕스패치 (Dang-spatch) - 유기견 구조 헬퍼 웹앱

## 1. 프로젝트 개요
사용자가 유기견을 발견했을 때, 현재 위치를 기반으로 가장 빠르게 관할 보호소에 연락을 취하고 당근마켓 동네생활에 도움 요청 글을 올릴 수 있도록 돕는 모바일 최적화 웹앱.

## 2. 기술 스택 (Tech Stack)
- Frontend: React (Vite 기반), JavaScript
- Styling: Tailwind CSS (모바일 반응형 UI 구성)
- Backend: 없음 (클라이언트 사이드에서 모든 API 호출 처리)

## 3. 핵심 기능 요구사항 (Core Features)

### 3.1 위치 파악 및 주소 변환 (Geolocation & Reverse Geocoding)
- 브라우저 Geolocation API를 통해 사용자의 현재 위도/경도 획득.
- 카카오맵 API(또는 구글 맵 API)를 활용해 위경도를 '도로명 주소' 텍스트로 변환하여 화면 상단에 노출.

### 3.2 주변 보호소 탐색 및 전화 연결
- 공공데이터포털 '동물보호센터 정보' API 호출.
- 내 위치를 기준으로 가장 가까운 보호소 3곳 리스트업 (보호소 이름, 관할 구역, 전화번호).
- '전화하기' 버튼 구현: `tel:` 링크를 사용하여 클릭 시 모바일 다이얼 앱으로 즉시 연결.

### 3.3 당근마켓 3초 컷 연동 (Deep Link & Clipboard)
- 화면 하단에 신고 텍스트 템플릿 자동 생성.
  - 템플릿 예시: "제목: [현재 동네] 유기견 보호/발견 / 내용: 방금 [도로명 주소] 근처에서 유기견을 발견했습니다."
- '내용 복사하고 당근마켓 열기' 버튼 구현.
- 클릭 시 작동 로직: 
  1. 위 텍스트가 기기 클립보드에 자동 복사됨.
  2. `daangn://` 커스텀 스킴(Deep Link)을 호출하여 당근마켓 앱 실행.

## 4. UI/UX 디자인 가이드
- 모바일 세로 화면(Mobile-first)에 최적화된 레이아웃.
- 당황한 상황에서도 직관적으로 누를 수 있도록 버튼을 크고 명확하게 디자인.
- 컬러 팔레트: 신뢰감을 주는 블루 계열 또는 따뜻한 오렌지 계열 포인트 컬러 사용.
